**splitwm** is a standalone X11 tiling window manager in Rust that replicates the terminal-multiplexer-style split layout of a Lua-based AwesomeWM config.

The architecture separates pure layout math (`tree.rs`, `state.rs`) from X11 event handling (`wm.rs`) and software rendering (`render.rs`). An arena-keyed n-ary tree (`NodeId` → `HashMap<NodeId, Node>`) keeps splits n-ary so collapsing a leaf redistributes its ratio proportionally to siblings rather than forcing a binary rebalance; `theme.rs` is the single source of truth for all palette and metric constants, while `state.rs` owns every mutation (split, close, resize, tab cycle, scroll, canvas width, insert-at-root, boundary resize, tab-move-to-sibling) so the event loop in `wm.rs` never touches the tree directly. Layout-changing actions trigger a wall-clock-driven ease-out-back animation (~280 ms) that interpolates frame rects with freshly created leaves starting from a collapsed sliver, with pre-split frame positions carried forward so content slides from its old spot. Frame windows are positioned/hidden by `arrange()` which dynamically computes canvas width from leaf count and minimum split width, maps off-screen frames out, reparents the active client into each frame, and resamples the focused client's top-center pixel content to adopt its accent colour. Gap boundaries expose draggable handles and "+" insert buttons (including edge insert buttons at the far left and right of the canvas) whose positions are updated each arrange pass, while a `smush` system synthesizes Ctrl+0/Ctrl+- keystrokes via XTEST to auto-shrink terminal fonts in narrow splits, cached per client and width bucket. A single full-screen underlay window composites the optional `SPLITWM_WALLPAPER` PNG, all leaf chrome, drag handles, and insert buttons below every client via chunked `PutImage`, avoiding any compositing-manager dependency, and tab bars use slanted trapezoid paths rendered with tiny-skia with per-tab app icons fetched from `_NET_WM_ICON` and letter-glyph fallbacks rasterized via fontdue.
